# Project Description

This project is a Brain Tumor Detection Web Application built using Python, Flask, TensorFlow/Keras, HTML, and CSS.
The system predicts whether a brain MRI image contains a tumor or not using a trained deep learning model.
the project can be integrated with Ollama Mistral AI model for generating intelligent explanations and user-friendly responses related to prediction results.

# Features

• Real-time Brain MRI image analysis

• Support for multi-class tumor detection

• Detect brain tumor using Deep Learning

• AI Chatbot for user interaction and medical query support

• AI-generated medical report based on prediction results
